## What this site is

A Next.js 14 / React 18 portfolio of 10 production SaaS platforms shipped solo over 5 years at Hashcash Consultants and as a personal developer build (Deal On Property). It includes:

- **Project case studies** at `/projects/<slug>` — written from the engineering perspective: problem, architecture, what shipped
- **Insights** at `/insights/<slug>` — interactive long-form writing on AI, architecture, and the messy parts of building production systems
- **Live visitor map** — real-time geo of who's reading, with admin notifications
- **Comment moderation** — every comment is reviewed by the admin before publish (no bots, no spam)

## Tech

Next.js 14 (App Router) · React 18 · TypeScript · MongoDB Atlas · Nodemailer · next-mdx-remote (v6, RSC) · gray-matter · Vercel.

The full insights engine — MDX pipeline, 10 interactive content components, behaviour tracking, HMAC-signed admin moderation, dynamic OG images, JSON-LD per post — is documented in `.claude/CLAUDE.md`.

## Selected work surfaced here

- **PayBito Integration MCP** — an AI agent knowledge server (Model Context Protocol) that injects PayBito's full API surface into any MCP-capable AI assistant. Published to PyPI + npm from one source of truth.
- **Multi-Tenant RAG Platform** — per-tenant private knowledge base, MongoDB Atlas vector search, cited retrieval.
- **Website Builder SaaS** (flagship) — no-code platform with end-to-end domain automation, multi-tenant DNS via AWS Route53, and a config-driven rendering engine.
- **BitoLink** — Slack-like real-time team communication with WebRTC → LiveKit SFU migration for group calls.
- **Paybito reCAPTCHA Service** — self-hosted captcha with siteverify-compatible API surface.
- **Deal On Property** — production real estate marketplace (Next.js 16 / React 19) built solo as the lead developer. Live at [dealonproperty.com](https://www.dealonproperty.com/) and on the [Google Play Store](https://play.google.com/store/apps/details?id=com.dealonproperty.mobile).
